# 🌱 El Viejo Manzano - Backend Ecommerce

Proyecto desarrollado en **Node.js + Express + MongoDB** como entrega final del curso de Backend.  
La aplicación permite gestionar productos, carritos y su visualización con imágenes y estilos básicos.

---

## 📌 Características principales

✅ Persistencia de datos con **MongoDB Atlas / Compass**  
✅ Gestión de **productos** con CRUD completo  
✅ Gestión de **carritos** con productos referenciados  
✅ Soporte para **paginación, filtros y ordenamiento** de productos  
✅ Renderizado con **Handlebars** para vistas dinámicas  
✅ Imágenes servidas desde la carpeta `public/uploads`  
✅ Estilos básicos aplicados a las vistas  
✅ API REST documentada para probar con **Postman**  

---

## 🛠️ Tecnologías utilizadas

- [Node.js](https://nodejs.org/)
- [Express](https://expressjs.com/)
- [MongoDB + Mongoose](https://mongoosejs.com/)
- [Handlebars](https://handlebarsjs.com/)
- [Socket.io](https://socket.io/) (opcional para comunicación en tiempo real)
- Git & GitHub
