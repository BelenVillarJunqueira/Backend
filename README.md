🌿 El Viejo Manzano - E-commerce 
Bienvenido al repositorio de El Viejo Manzano , una tienda online creada con JavaScript y Handlebars, que permite explorar un catálogo de productos relacionados con jardinería, plantas, césped, macetas y herramientas.

Este proyecto forma parte de una práctica académica con el objetivo de aplicar los conceptos fundamentales de Backend.

🚀 Funcionalidades
🛒 Vista principal con el catálogo de productos.
📂 Filtro por categorías dinámicas.
🔍 Vista en detalle de cada producto.
➕ Selector de cantidad en la vista de detalle.
❌ Ruta 404 para navegación no válida.
